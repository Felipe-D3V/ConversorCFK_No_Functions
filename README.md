Um conversor simples de temperaturas entre **Celsius**, **Fahrenheit** e **Kelvin**.

##  Funcionalidades
- Converte Celsius ↔ Fahrenheit ↔ Kelvin
- Interface simples via terminal
- Código escrito em C
